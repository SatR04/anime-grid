# 🏮 The Anime Hall of Fire
**A Cinematic 3D Showcase Collection**

Live Demo: [https://anime-grid-two-phi.vercel.app](https://anime-grid-two-phi.vercel.app)

## ✦ Overview
This project is a high-performance, responsive anime showcase built with React and Vite. It features a dark, luxury aesthetic inspired by Japanese editorial design, utilizing 3D perspective transforms and real-time data fetching.

## ✦ Key Features
- **3D Interactive Grid:** Cards respond to mouse movement with dynamic tilt and zoom.
- **Cinematic Motion:** Staggered scroll-reveal animations using Intersection Observer.
- **Live Data:** Fetches official high-res posters and metadata via the Jikan (MAL) API.
- **Adaptive Layout:** Fluid 2-column to 1-column transition for mobile/tablet optimization.

## ✦ Tech Stack
- **React 18** (Functional Components & Hooks)
- **Vite** (Next-gen frontend tooling)
- **CSS3** (Custom Properties, Perspective, & Keyframe Animations)
- **Git/Vercel** (Continuous Deployment)
